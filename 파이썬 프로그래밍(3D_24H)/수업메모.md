## input 사용 Tip

숫자처럼 계산할 값이면 int()나 float()로 감싸서 형 변환하고,
그 외(문자·이름·문장 등)는 그냥 input()으로 문자열로 받으면 됨.
`int = 정수`
`float = 실수`

**예시**
```py
# 이름
input_name = input("이름을 입력하세요: ")
int(input("나이를 입력하세요: "))
```
