### 코드에서 하는 일 순서

**두 행렬 정의**

```py
A = np.array([[1, 2],
              [3, 4]])
B = np.array([[5, 6],
              [7, 8]])
```

**요소별 곱셈 (element-wise multiplication)**
각 위치별로 곱함:

```py
[[1*5, 2*6],
 [3*7, 4*8]] → [[5, 12], [21, 32]]
```

**행렬 곱셈 (dot product)**

```py
np.dot(A, B)
[[1*5 + 2*7, 1*6 + 2*8],
 [3*5 + 4*7, 3*6 + 4*8]] → [[19, 22], [43, 50]]
```

**전치행렬 (Transpose)**
행과 열을 바꾼 형태:

```py
A.T = [[1, 3],
       [2, 4]]
```

**역행렬 (Inverse)**

```py
A_inv = np.linalg.inv(A)
[[-2. , 1. ],
 [ 1.5, -0.5]]
```

**검증 (단위행렬 확인)**
```py
np.dot(A, A_inv)
[[1, 0],
 [0, 1]]
```
>진짜 역행렬이 맞다는 증명.
