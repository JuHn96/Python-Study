## 💡 핵심 요약

### 꼭 기억할 것:

1. **리스트 슬라이싱**: `data[start:end:step]`
2. **리스트 컴프리헨션**: `[x**2 for x in range(10) if x % 2 == 0]`
3. **딕셔너리**: key-value 쌍, `.get()`, `.items()`
4. **함수**: 재사용 가능한 코드 블록
5. **람다**: 간단한 일회용 함수

### 연습 팁:

- 리스트 컴프리헨션은 처음엔 어렵지만, 매우 자주 사용됩니다!
- 딕셔너리는 데이터 저장/조회에 필수입니다
- 함수로 코드를 정리하는 습관을 들이세요
- 초급 → 중급 → 고급 예제를 순서대로 풀어보세요
